Notes About PraiseHub & wws Rails App

These are quick notes about where and how pages and stuff is done in the wws Rails App for PraiseHub. Where Stuff Is: dashboard => wws/app/views/home/dashboard.html.erb sign in => wws/app/views/devise/sessions/new.html.erb sign up => wws/app/views/devise/ forgot password => wws/app/views/devise/passwords/new.html.erb reset password => wws/app/views/devise/passwords/edit.html.erb confirm password => wws/app/views/devise/confirmations/new.html.erb mailer confirmation instructions => wws/app/views/devise/mailer/confirmation_instructions.html.erb mailer reset password instructions => wws/app/views/devise/mailer/reset_password_instructions.html.erb mailer unlock instructions => wws/app/views/devise/mailer/unlock_instructions.html.erb What It Needs: Ruby 1.8.7, RubyGems 1.8.15, Rails 3.2.9, Apache2, MySQL, Passenger, etc. Restart, Start & Stop: touch wws/tmp/restart.txt or sudo service apache2 restart (or start or stop)
